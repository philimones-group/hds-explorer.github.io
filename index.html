The power of demographics surveillance open-sourced.

HDS-Explorer its a server-client application designed from its core provide to be a free and open-source data collection and data repository for health and demographics surveillance system.
The application can be found on public repository on Github and through Google Play Store.

Some of the Features

Id Generator Customization

Allow easy implementation of a site-level Id schemes customization, enables each site to reuse their previous codes on Regions, Households and Members.

On Mobile Attributes Editor

Supports editing Region, Households and Members atributtes on Mobile app and synhcronize later the edited datasets with the server.

Web-Based Validation Error Editor

Using the Server web-app, its possible to visualize errors and edit records with correct values and synchronize the data.

Ease Of Use

With a User Friendly UI, you can easily navigate through the HDSS datasets, and perform multiple research forms data collection.

ODK Forms Management

Create Form deﬁnitions and use HDSS
data to preload data to ODK Forms.

Form Groups

Ability to group ODK Forms as one (Allow to split large forms into smaller and mantaining a link between them).

Fast Access To HDSS

Fast access to explore demographics surveillance data powered by a light NoSQL Database in the Mobile App.

External Datasets

Ability to visualize studies datasets on Mobile App and pre-populate data into Electronics Forms (ODK).

Modular Data Access

Grant access on data (Region, Household, Member, Forms) to a speciﬁc group of Users (that belongs a study).

Mobile Ofﬂine Data Sharing

Pre-register Regions or Households and share the datasets with other near devices (helps to avoid duplications).

Case study: malawy

Intro
Method
Results

- Health and Demographics Surveillance
Systems (HDSS) allow accurate electronic
data capture.

- HDSS facilitate the understanding of local
community health issues.

- Widely used HDSS do not work well in
resource-limited settings.

- HDS Explorer works offline, supports
multiple Android versions.

- Further, it seamlessly integrates with any
version of Open Data Kit (ODK).

- Aimed to assess usability of the HDS
Explorer in resource-limited settings in
Southern Malawi.

Evaluated system benchmarks:

- Synchronization speed
- Data compression
- Data validation accuracy
- Ability to manage questionnaire
- Interoperability
- Manages participant follow-ups.


The system was assessed in the managing of:

- Cross-sectional census
- Surveillance studies capturing about 6,000 individual health center visits
- 10 community and hospital-based cohort studies including 102,804 subjects.

- HDS Explorer integrated with ODK and prevents missed and repeated eCRFs and missed follow-up visits.

- Synchronization was fast. Validation scripts prevented capture and synchronization of incorrect data.

- It was easy to learn.

- Thanks to its data compression algorithms it keeps information of key demographic events throughout one’s life span.

- Media files allowed real-time queries based on longitudinal data, for accurately capturing anthropometric measures.
